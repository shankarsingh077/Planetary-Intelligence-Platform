# Data Source Expansion Plan

Goal: Scale from 4 prototype connectors to 50+ managed sources with quality controls.

## Wave Plan

## Wave 1 (Weeks 1-3): Reach 12 sources

Domains:
- Geopolitics
- Markets
- Regulation
- Mobility

Deliverables:
1. 8 additional connectors beyond current 4.
2. Source SLAs declared and monitored.
3. Provenance and explainability mapping validated.

## Wave 2 (Weeks 4-8): Reach 25 sources

Domains:
- Climate
- Cyber
- Aviation
- Maritime

Deliverables:
1. Connector deployment templates by source class.
2. Per-source quality score dashboards.
3. DLQ replay automation for all P0/P1 connectors.

## Wave 3 (Weeks 9-14): Reach 40 sources

Domains:
- Energy
- Supply chain
- Humanitarian
- Health

Deliverables:
1. Entity enrichment mappings per domain.
2. Cross-domain fusion readiness checks.
3. Regional resiliency and redundancy strategy.

## Wave 4 (Weeks 15-20): Reach 50+ sources

Domains:
- Narratives/social
- Satellite
- Specialized policy feeds

Deliverables:
1. Source trust index baseline.
2. Narrative contradiction tracking.
3. Scrape fallback governance controls.

## Quality gates for each wave

1. Freshness SLO attainment >= 95% on P0/P1 sources.
2. Provenance completeness == 100%.
3. Contract test pass rate == 100%.
4. No unresolved legal/compliance blockers.
