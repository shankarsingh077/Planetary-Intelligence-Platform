# PIP Presentation

Files:

- `judge-presentation.html` - full-screen browser presentation
- `judge-presentation-notes.md` - speaker script and Q&A prompts
- `assets/` - project visuals used in the deck

How to use:

1. Open `judge-presentation.html` in a browser.
2. Use `Left` and `Right` arrows or `Space` to navigate.
3. Press `F` for full screen.
4. Press `N` to toggle presenter notes inside the deck.

The deck uses real project visuals from the repository and does not modify the application code.
